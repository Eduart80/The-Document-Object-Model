# Document Object Model

App MyBlog. 
Programming language: HTML, CSS, JavaScript

Signup page instruction: 

Enter user name, email, and create a new password, and then re-enter the same password for verification.<br>
The data will be saved on local storage. After confirmation, the user will be redirected to a blog page.<br>
On the right side, the user has an input area to enter the text of choice.<br>
The tag From: will be populated by the user name that you entered in the signup form
This data will be retrieved from local storage.
Users have an input field to enter the title and a text area to write any remarks. On the submit button,
a new card will be created, with the user name, title, and the text.
The user will be able to create new cards on every click of the submit button.
